BerriosPay

Enterprise payment kiosk system for Empresas Berrios Inc. — full hardware/software integration with resilience-first architecture.

Architecture Overview

Presentation Layer
React 19 + TypeScript + TailwindCSS
Kiosk Shell
WPF + WebView2 (Hardware Bridge)
API Layer
.NET 10 Minimal APIs + Clean Arch
Data + Cloud
Azure SQL + App Service + Bicep IaC

The Challenge

Build a payment kiosk system that integrates a PAX A920 payment terminal, Epson TM-M30II thermal printer, and Touch Dynamic QK22 kiosk into a unified, bilingual, WCAG 2.1 AA accessible platform — designed to operate reliably even when network connectivity is intermittent.

Architecture Decisions

  • Polly circuit breaker + retry for hardware resilience
  • Idempotency layer preventing duplicate payments
  • Offline queue with automatic sync recovery
  • WPF shell bridging web app to native hardware

Sole Engineer

  • Designed and built the entire system solo — frontend, backend, kiosk shell, database schema, cloud infrastructure, and hardware integration layer.

Tech Stack

React 19 .NET 10 WPF + WebView2 Azure SQL TailwindCSS Polly
