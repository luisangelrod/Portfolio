Entry #004 — March 23, 2026 AI Engineering

I Built an AI Agent to Write My Tests — It Now Knows My Codebase Better Than I Do

There's a specific kind of developer pain that comes from staring at a feature spec and knowing you need to write 40 Vitest tests before any of it ships. I solved it by building AgentQA — an agentic tool that takes a feature spec and streams a complete, type-safe test suite back to you in real time.

The core is Claude's tool_use API. I defined a single tool — generate_vitest_case — and let Claude call it once per requirement extracted from the spec. Each invocation returns a fully runnable Vitest test with realistic mocks, type annotations, and edge cases that I wouldn't have thought to write myself. The results stream back over SSE so you see tests appearing live as the agent works.

The result: CDD-HUB now has 215 tests. About 40 of them were generated by AgentQA in a single session. The agent caught a duplicate-transaction edge case I'd missed — the kind that only surfaces at 2 AM in production. It didn't. Because the agent wrote the test first.

This is what "AI engineering" actually means in practice. Not using ChatGPT to autocomplete code. Building tools that extend your own leverage — agents that work in the gaps your attention can't cover.

Entry #003 — March 19, 2026 WebGPU Renderer

Why I Chose WebGPU Over a Traditional Game Engine

The obvious choice for a browser-based AAA-quality game would be Unity WebGL or Godot. I considered both for about a week, then abandoned them for a reason that sounds insane: I want to own the renderer. Not wrap it. Not configure it. Own it.

WebGPU gives you direct access to the GPU pipeline — compute shaders, render passes, bind groups — without the abstraction tax of a full engine. For ECHO's visual identity (deep underwater SSAO, volumetric light shafts through flooded corridors, PBR materials with custom wetness shaders), that level of control isn't optional. It's the whole game.

Current status: Three.js WebGPU renderer is bootstrapped. I've got a basic scene with PBR materials and a prototype of the depth-of-field post-processing pass. The screen-space ambient occlusion shader is half-written and causing the exact kind of pain that means I'm learning the right things.

Entry #002 — March 12, 2026 Game Concept

ECHO: The World, The Tone, The Why

ECHO is set in an abandoned deep-sea research station, 3,000 meters below the Atlantic. You're the first human to return to Facility Kilo-7 after a catastrophic communication blackout 11 years ago. No combat. No weapons. Just exploration, environmental storytelling, and the growing certainty that something is still down here with you.

The gameplay loop is: exploration → audio logs / environmental clues → puzzle-unlocked zones → escalating dread. Think Soma meets SOMA's atmosphere with the environmental storytelling of Outer Wilds. I want the water physics to feel oppressive. The acoustics to feel wrong. The light to feel rationed.
