# CLAUDE.md

## Project Overview

Static portfolio for Luis Rodriguez, positioned for senior engineering and systems roles.

Primary goals:

- present shipped production work quickly
- support recruiter scanning in one screen
- keep personal branding aligned to `luisangelrod`

## Stack

- HTML5
- Tailwind CSS via CDN
- Vanilla JavaScript
- Lucide icons
- Netlify Forms

## Local Development

Use any static server:

```bash
python -m http.server 8000
```

or

```bash
npx serve
```

## Key Files

- `index.html` - full site
- `assets/LA_Resume_.docx` - downloadable resume
- `img/projects/` - project screenshots
- `robots.txt` and `sitemap.xml` - SEO support

## Content Rules

- prioritize production systems over speculative side projects
- avoid generic AI buzzword-heavy messaging
- keep copy direct, recruiter-readable, and outcome-focused
- use personal GitHub links only: `https://github.com/luisangelrod`
