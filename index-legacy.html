    Technologies

    A list of my skills and languages and I utilize.

    HTML 5

    HTML is the foundation of every website and web application. This markup language tells web browsers how to display a web page's words and images.

    CSS 3

    CSS is a language for describing the way web pages are presented, including colors, layout, and fonts.

    JavaScript

    Javascript is a lightweight, object-oriented language best known as the scripting language for web pages. It provides web pages with their functionality allowing users to interact with the page.

    C#/.NET 5

    C# is an enterprise level object-oriented programing langage that enables developers to build a variety of applications from desktop to web pages and mobile applications. Paired with the .NET framework, it is a very powerful tool.

    Database Design & SQL

    Relational database design, and using SQL for queries and data management. This technology is critical to most serious projects.

    Bootstrap

    Bootstrap is a leading design framework that allows for responsive and mobile-ready development. Based on a standardized grid layout, the Bootstrap framework allows for professional and responsive applications to be created quickly and easily.
